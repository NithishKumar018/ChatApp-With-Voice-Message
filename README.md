💬 Real-Time Chat Application (MERN + Socket.io)

A full-stack, production-ready real-time chat application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) and Socket.io. This app supports one-to-one chat, group chat, typing indicators, voice messages, file sharing, and online/offline presence tracking, similar to WhatsApp Web or Discord.
